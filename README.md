## Skill Set

### UI/UX Development
Building user interfaces that are friendly, simple, useful, and scalable. Focus on intuitive design patterns and consistent user experiences across all devices and platforms.

### Accessibility
WCAG 2.1 AA/AAA compliance, semantic HTML, ARIA attributes, keyboard navigation, screen reader optimization, color contrast analysis, focus management, and accessible forms.

### Performance Optimization
Core Web Vitals optimization including LCP (Largest Contentful Paint), FID (First Input Delay), CLS (Cumulative Layout Shift), TTFB (Time to First Byte), and INP (Interaction to Next Paint). Code splitting, lazy loading, image optimization, caching strategies, and bundle size reduction.

### Web Discoverability
SEO (Search Engine Optimization), PPC (Pay-Per-Click) campaign integration, GEO (Generative Engine Optimization), and AEO (Answer Engine Optimization). Structured data, meta optimization, sitemap management, and search console analysis.

### Security
OWASP Top 10 awareness, XSS/CSRF prevention, Content Security Policy, secure authentication flows, input sanitization, HTTPS enforcement, dependency vulnerability scanning, and penetration testing preparation.

### Audits
Experience conducting comprehensive audits covering accessibility, performance, SEO, and security. Lighthouse, axe DevTools, WebPageTest, GTmetrix, and custom audit tooling.
